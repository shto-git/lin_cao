# 项目推进记录

## 2026-06-25（续）— 核心功能实现

### 本轮完成

1. **补全 RAGFlow 客户端**（`ragflow_client.py`）：
   - 新增：创建/列出/删除知识库
   - 新增：上传文档（multipart/form-data）、列出/删除文档
   - 新增：停止解析、创建对话助手、OpenAI-compatible 对话补全
   - 重构：统一 `_send()` 底层方法，代码更简洁

2. **实现章节草稿生成模块**（`generator.py`）：
   - `LLMClient`：支持任意 OpenAI-compatible API（DeepSeek/Qwen/OpenAI 等）
   - `LLMConfig`：从环境变量读取配置
   - `generate_chapter_draft()`：RAG-enhanced prompt 模板 + 占位草稿降级
   - 无 API Key 时自动返回结构化占位草稿（含证据摘要）

3. **实现完整 Pipeline 编排**（`pipeline.py`）：
   - `run_pipeline()`：outline → retrieval → draft → quality → export 一键执行
   - 自动导出 4 个文件：outline.md、retrieval_plan.md、document.md、quality_report.md
   - 支持 RAGFlow 检索 + LLM 生成 + 全链路自动化

4. **补全渲染器**（`renderer.py`）：
   - 新增 `render_full_document()`：目录 + 草稿正文 + 质检报告一体化输出
   - 新增 `render_chapter_draft()`：单章节草稿渲染

5. **新增 Word 导出**（`word_export.py`）：
   - Markdown → .docx 转换
   - 支持标题层级、列表、引用、首行缩进

6. **完善 CLI**（`cli.py`）：
   - 支持 `--skip-llm` 模式（仅大纲+检索计划）
   - 支持 `--type` 覆盖规划类型
   - 完整流程模式（含章节生成和质检）

7. **测试覆盖**：
   - 23 个单元测试全部通过
   - 覆盖：原有 3 个 + 模板扩展 + 生成器 + 渲染器 + RAGFlow 客户端 + 完整 Pipeline
